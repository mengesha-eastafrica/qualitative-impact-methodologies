# Non-Linear Qualitative Impact Frameworks

## Operationalizing Most Significant Change (MSC) & Outcome Mapping in Fragile Contexts
**Author:** Mekuria Abera Mengesha  
**Application Domain:** Governance, Rule of Law, and People-Centred Justice Systems

### 1. The MSC Story Selection Protocol
This repository demonstrates a structured, participatory workflow designed to capture emergent, unplanned outcomes where traditional linear indicators fail to measure systemic shifts.

   [Field Collection] ──> Local staff gather raw Stories of Change
           │
           ▼
 [First-Level Review] ──> Regional Project Selection Panel (FGD)
           │
           ▼
[Institutional Review] ──> Strategic Steering Committee Validation
           │
           ▼
[Feedback Loop Execution] ──> Insights update Theory of Change (ToC)


### 2. Story Harvesting & Metadata Schema
To ensure absolute data quality assurance and verification readiness for international bilateral donors (e.g., Dutch MFA), every harvested narrative is processed using the following structured metadata matrix:

| Evaluation Variable | Data Field Specification | Operational Control Objective |
| :--- | :--- | :--- |
| **Contextual Baseline** | Description of initial institutional/social environment | Establishes the pre-change environment boundaries. |
| **The Intervention Node** | Concrete action or programmatic element introduced | Isolates the catalyst within the wider system. |
| **The Narrative Core** | Direct testimony documenting the specific shift | Captures qualitative human-centric impact evidence. |
| **Significance Justification** | Systematic explanation of *why* this change matters | Drives the participatory learning analysis loop. |
